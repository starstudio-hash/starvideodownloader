Install on Mac

  1. Download the latest ZIP from the website or GitHub release.
  2. Open the ZIP and drag Star Video Downloader into Applications.
  3. Open the app and let it install yt-dlp and ffmpeg if prompted.

If macOS says the app is damaged or cannot be opened

Delete the old downloaded ZIP and app first, then download the latest release again. If Gatekeeper still blocks it, move the app to Applications and use Open Anyway in System Settings > Privacy & Security.

If you are comfortable with Terminal and trust this app, this also removes the quarantine flag:

xattr -dr com.apple.quarantine "/Applications/Youtube downloader.app"

Activate Pro

  1. Buy Pro through Gumroad.
  2. Copy the license key from the email receipt.
  3. Open the app, go to Settings, paste the key, and click Activate.

The Settings screen now shows your Free or Pro status, the last validation result, and a manual Validate License Now action. If you are moving to another Mac, use Remove from This Mac on the old install first so your local activation is cleared there.

Send Better Support Details

If a download, activation, conversion, or repair job fails, open Settings and choose Export Support Bundle. That file includes backend health checks, queue state, license status, and recent failures without including your full license key.

Email the exported JSON to support@starvideoapp.com with the URL or file you were working with and a short note about what happened.
