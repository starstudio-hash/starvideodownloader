March 11, 2026

Workflow, reliability, and release hardening update
  • Added always-visible Free / Pro status in the main app header.
  • Added URL preflight so the app can inspect title, channel, duration, live status, and playlist details before enqueue.
  • Added selectable playlist import instead of forcing “download all.”
  • Improved duplicate detection so common tracking/query variations do not create accidental duplicates.
  • Improved download path tracking to reduce wrong-file attachment after completion.
  • Added startup license validation, clearer license state messaging, and a manual validate action.
  • Added support-bundle export from Settings for faster troubleshooting.
  • Added persistence for download, conversion, and repair queues across relaunches.
  • Updated the website to use the latest GitHub release download URL instead of one hardcoded tag.
